Conseguimos compilar e executar com:
  gcc (Ubuntu 9.4.0-1ubuntu1~20.04.1) 9.4.0
  gcc (Ubuntu 11.2.0-19ubuntu1) 11.2.0

Como compilar:
  make

Como executar:
  ./main <arquivoDeImagem>

Caso queira sair da shell use o comando: exit
